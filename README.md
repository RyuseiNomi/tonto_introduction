## 主な技術スタック

* Typescript v4.4.2
* React v18.1.0
* yarn v1.22.4

## ローカル環境の立ち上げ

以下のコマンドを実行し、`localhost:3000` へアクセスする

```
$ yarn start
```

`yarn`が無い場合は、以下の手順でインストールを行う

```
$ npm install -g yarn
```

## テストの実行

プロジェクトのルートディレクトリで以下を実行する

```
$ yarn test
```
